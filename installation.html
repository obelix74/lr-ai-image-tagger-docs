System Requirements

Before you begin, ensure you have:
  • Adobe Lightroom Classic 2024 or newer
  • Internet connection for AI analysis
  • Google account for API key (free)
  • Windows 10/11 or macOS 10.15+

Step 1: Download Plugin

1 Download the Latest Version

Download the Gemini AI Image Tagger plugin from our secure download link:

Download Gemini AI Image Tagger v2.3.1

File size: ~50KB | Version: 2.3.1 | Updated: January 2025

2 Extract the Plugin
  1. Locate the downloaded ZIP file (usually in your Downloads folder)
  2. Right-click the ZIP file and select "Extract All" (Windows) or double-click (macOS)
  3. Choose a location to extract (e.g., Desktop or Documents)
  4. You should see a folder named "Gemini AI Image Tagger" containing the plugin files

Step 2: Get Gemini AI API Key

1 Visit Google AI Studio

Go to https://ai.google.dev/gemini-api/docs/api-key

2 Sign In with Google

Use your existing Google account or create a new one. No credit card required for the free tier.

3 Create API Key
  1. Click the "Get API Key" button
  2. Select "Create API key in new project" (recommended for first-time users)
  3. Your API key will be generated instantly
4 Copy and Secure Your API Key
Important:
  • Copy the API key immediately (it looks like: AIzaSyA...)
  • Store it securely - treat it like a password
  • Never share it publicly or commit it to version control
Free Tier Limits:
  • 15 requests per minute - Perfect for normal photo editing workflows
  • 1,500 requests per day - Analyze hundreds of photos daily
  • 1 million tokens per month - More than enough for most users

These limits are generous for most photography workflows. If you need more, paid tiers are available.

Step 3: Install in Lightroom

1 Open Lightroom Classic

Launch Adobe Lightroom Classic on your computer.

2 Access Plug-in Manager

Go to File > Plug-in Manager in the menu bar.

3 Add the Plugin
  1. Click the "Add" button in the Plug-in Manager
  2. Navigate to the extracted "Gemini AI Image Tagger" folder
  3. Select the folder and click "Select Folder" (Windows) or "Choose" (macOS)
  4. The plugin should appear in the list as "Gemini AI Image Tagger"
4 Verify Installation
Success! You should see "Gemini AI Image Tagger" in the plugin list with a status of "Installed and running".

Step 4: Configure Plugin

1 Select the Plugin

In the Plug-in Manager, click on "Gemini AI Image Tagger" to select it.

2 Enter API Key
  1. Find the "Gemini AI API Key" section
  2. Paste your API key in the "API Key" field
  3. The synopsis should change to "API key configured"
3 Configure Settings (Optional)

Customize these settings based on your workflow:

  • IPTC Metadata: Choose which fields to save automatically
  • Keyword Options: Set how keywords are created and organized
  • Batch Processing: Adjust batch size and delays for your internet speed
  • AI Prompt: Use default or create custom analysis prompts
4 Save Configuration

Click "Done" to save your settings and close the Plug-in Manager.

Step 5: First Use

1 Select Photos

In Lightroom's Library module, select one or more photos you want to analyze.

2 Start AI Analysis

Go to Library > Plug-in Extras > Tag Photos with AI

3 Review Results

The AI analysis dialog will show:

  • Caption: Brief description of the photo
  • Description: Detailed description
  • Keywords: Relevant tags (select/deselect as needed)
  • Instructions: Usage suggestions
  • Copyright: Attribution information (if detected)
  • Location: Identified landmarks
4 Apply Metadata

Click "Apply" for the current photo or "Apply All" for all analyzed photos.

Troubleshooting

Common Issues and Solutions

Problem: "API key not configured" error
Solution:
  • Verify you've entered the correct API key in plugin settings
  • Ensure there are no extra spaces before or after the key
  • Try generating a new API key if the current one doesn't work
Problem: "Network error" during analysis
Solution:
  • Check your internet connection
  • Verify firewall isn't blocking Lightroom
  • Increase delay between requests in plugin settings
  • Try analyzing fewer photos at once
Problem: Plugin doesn't appear in menu
Solution:
  • Restart Lightroom Classic
  • Check that plugin status is "Installed and running"
  • Try removing and re-adding the plugin
  • Ensure you selected the correct plugin folder
Problem: Analysis takes too long or times out
Solution:
  • Reduce batch size in plugin settings
  • Increase delay between requests
  • Analyze photos during off-peak hours
  • Check if you've exceeded API rate limits

What is Glean?

Glean is a professional-grade, native macOS Digital Asset Management (DAM) application designed for photographers, photojournalists, and archivists. It combines blazing-fast culling workflows with powerful cataloging and AI-driven automation features to streamline your workflow.

Key Features

  • Uncompromising Speed: Optimized for instantaneous response times, from loading thousands of RAW images to navigating previews.
  • Intelligent Automation: AI-powered metadata generation, keyword tagging, and natural language search (powered by Gemini, OpenAI, or local models).
  • Advanced Duplicate Detection: Find near-duplicates with perceptual hashing and semantic duplicates with a local CLIP model.
  • Robust Cataloging: Manage millions of photos across dozens of online and offline drives with a powerful SQLite-based backend.
  • Seamless Workflow: Designed for fluid integration with tools like Adobe Lightroom, Capture One, and Photoshop.
  • Comprehensive Metadata Support: Hybrid engine using Apple's Image I/O for speed and ExifTool for deep, comprehensive metadata control.

Installation & Setup

✅ Ready to Use! Glean works immediately after installation - no additional setup required. Simply download the DMG, drag to Applications, and start organizing your photos!

What's Included?

Glean uses Apple's Vision Framework built into macOS for fast, accurate duplicate detection. This means:

  • ✅ Zero configuration - works right out of the box
  • ✅ Fast performance - optimized for Apple Silicon and Intel Macs
  • ✅ Small download - only 6.7 MB
  • ✅ No internet required - everything runs locally

Duplicate Detection Technology

Glean uses Apple's Vision Framework by default for duplicate detection, which provides:

  • ⭐⭐⭐⭐⭐ Excellent duplicate detection - finds near-identical photos with different edits, crops, or filters
  • ⚡ Fast performance - processes ~50ms per image
  • ✅ Built into macOS - no additional downloads needed

Bundled: Apple MobileCLIP for Semantic Similarity

Find all photos of "beaches" or "mountains" regardless of how they look with the bundled MobileCLIP model for semantic similarity!

✅ Included in the App

Glean includes Apple's MobileCLIP S0 model (~103 MB) for semantic similarity and advanced content-based searches. No additional downloads or setup required!
